Manually and Automatically Calculating gradients

What exactly is requires_grad? - Allows calculation of gradients w.r.t the tensors that all allows gradients accumulation.

yi=5(xi+1)2y_i = 5(x_i+1)^2yi​=5(xi​+1)2﻿

# Create a tensor of size (2,1) and requires gradients
x = torch.ones(2, requires_grad = True)
print(x)


# Simple Linear Equation with the tensor x
y = 5 * (x + 1) ** 2

print(y)
🤙
tensor([ 1.,  1.])  # x

tensor([ 20.,  20.]) # y

Note: Backward should be called only on scalar(i.e.,1-element tensor) or w.r.t the variable

Let's reduce the y into a scalar then:

o=12∑iyio = \frac{1}{2}\sum_i y_io=21​∑i​yi​﻿

o = (1/2) * torch.sum(y)

o

Calculating First Derivative:

Recap y equation: yi=5(xi+1)2 y_i = 5(x_i+1)^2 yi​=5(xi​+1)2﻿

Recap o equation: o=12∑iyio = \frac{1}{2}\sum_i y_io=21​∑i​yi​﻿

Substitute y into o equation: o=12∑i5(xi+1)2o = \frac{1}{2} \sum_i 5(x_i+1)^2o=21​∑i​5(xi​+1)2﻿

∂o∂xi=12[10(xi+1)]\frac{\partial o}{\partial x_i} = \frac{1}{2}[10(x_i+1)]∂xi​∂o​=21​[10(xi​+1)]﻿

∂o∂xi∣xi=1=12[10(1+1)]=102(2)=10\frac{\partial o}{\partial x_i}\bigr\rvert_{x_i=1} = \frac{1}{2}[10(1 + 1)] = \frac{10}{2}(2) = 10∂xi​∂o​∣∣∣​xi​=1​=21​[10(1+1)]=210​(2)=10﻿

And it's so simple with PyTorch with the following line.

o.backward()
